# Tools

Personal automation tools and scripts for administrative tasks.

## Available Tools

### [Timesheet Generator](./timesheet-generator/)
Generate professional PDF timesheets with customizable parameters for employee time tracking.

- Multi-language support (Spanish/Portuguese/English)
- Configurable date ranges
- Automated weekday calculation
- Print-ready format

## Installation

Fork this repository and clone it to your local machine:

```bash
# Fork on GitHub first, then:
git clone https://github.com/YOUR_USERNAME/tools.git
cd tools
```

Navigate to the specific tool directory for usage instructions.

## Repository Structure

```
tools/
├── timesheet-generator/    # PDF timesheet generation tool
└── kids-project            # Educational materials generator for school projects
└── [future tools]          # Additional utilities to be added
```
